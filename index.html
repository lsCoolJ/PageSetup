    The Raspberry Pi is a compact computer that can be used for many, many cool things such as your own personal network storage device, a personal music player, a retro video game player, etc. Setting up the raspberry pi is relatively easy, too, which makes using one very practical.

    Setup

    What you need

    1. A Raspberry Pi (Duh) You can find raspberry pis online for around 35 to 40 dollars. It doesn't particularly matter which raspberry pi you get, as long as it feeds your needs. One of the most common pis to get is the Model B:

    2. A Power Source. You will need an AC adapter with a USB micro cable. Make sure the AC Adapter can power 5V.

    3. An SD Card with at least 4 GB (recommended 8GB) of memory. I recommend a micro SD card so you can use it with both a B and B+ Raspberry Pi.

    4. A computer monitor or TV (with HDMI).

    5. A usb keyboard and mouse. I suggest a keyboard that has the mouse track pad built into it so that you're only taking up one usb port on your raspberry pi.

    6. HDMI Cable. You can get an HDMI to HDMI cable or an HDMI to DVI cable, whichever fits your needs.

    7. A Network connection. You can either use a Wifi Dongle, or an Ethernet cable connected to a wireless bridge. A network connection is important to have in case you want to install software on the pi or if you want to access your pi remotely.

    8. Speakers are optional.

    Flashing the SD card with an image.

    Note: If you don't want to go through the process of flashing your own SD card, you can purchase a pre-loaded one off Amazon or other websites.

    The Raspberry Pi typically runs on a form of Debian (a linux-like operating system). The first step of getting your Raspberry Pi to run is to put this image on the SD card so that your Pi can actually boot up. While your Mac, or PC, uses a magnetic hard disc drive (HDD) or a Solid State Drive (SSD), your Raspberry Pi has no internal storage of its own.

    Instead, the Raspberry Pi uses a Secure Digital (SD) card, like the ones that are used for storage in digital cameras.

    This means that the Raspberry Pi will run from whatever operating system you install on an SD card. You can easily flash the SD card with an operating system on your Mac or PC. Here's how:

    Using OS X

    Format the card

    Formatting an SD card is quite straight forward with Mac OS X. Start by pressing CMD SPACE to reveal the Spotlight search bar. In this search bar, type "disk utility" and press return to launch the Disk Utility application.

    Select the SD card, in the left hand pane, and then select Erase ensuring that the format is "MS-DOS (FAT)". If the format is already MS-DOS (FAT), then you have nothing to worry about.

    Identify the card

    Open Terminal. If you need help finding this, press CMD SPACE as you did above and type in "Terminal" and press return. Type in diskutil list to get a list of the disks your computer is currently using. You should see this in your terminal:

    Make sure you know which disk is your SD card. For example, mine is /dev/disk1. If you use the wrong disk, you'll end up messing up your computer pretty badly.

    Next, you'll need to unmount your disk if it is automatically mounted. To do this, type in the terminal diskutil unmountDisk /dev/disk1. Be sure to change the disk number if you identified yours to be different.

    Downloading an image

    Now you'll want to download the image you want to put on your Pi. http://downloads.raspberrypi.org/ has many images to choose from. I personally chose the basic Raspbian image located at the top of this page: http://downloads.raspberrypi.org/raspbian/images/raspbian-2015-02-17/. You may, however, choose whichever image fits you best.

    Flash the card

    Now that you have an image downloaded, you'll want to navigate to the folder where you downloaded and extracted the image. This will typically be your Downloads folder if you did not change the location or extract the folder in a different place. If you haven't extracted your image yet, then simply open up Finder, navigate to your downloads folder and open the zip file you just downloaded.

    Navigate to the folder by typing cd Downloads into your terminal.

    Now we are going to use the dd command in terminal. This command is very powerful and you risk serious corruption to your Mac if you use it incorrectly. Triple check your command before executing it.

    Ensuring that you replace the .img and destination, in my example, with the correct .img name destination disc for yours, enter the following command:

    sudo dd if=2015-02-16-raspbian-wheezy.img of=/dev/disk1 bs=2m

    This process can take a few minutes, so go ahead and take a break. Go to the bathroom or get a snack. Once the process is complete, you should see a few lines in the terminal indicating records were transferred and bytes were moved. For example:

    925+0 records in
925+0 records out
19398665600 bytes transferred in 576.382578 secs (3365587 bytes/sec)

    You are now free to unmount the SD card for use on your Raspberry Pi

    Using Linux

    The process is very similar to the OS X process, although there is a minor difference when identifying the card. When you want to identify the card, simply type sudo fdisk -l. The rest of the process is the same.

    Using Windows

    Format the card

    Locate your SD card drive, in Windows Explorer, and right-click the mouse to bring up the context-senstive menu. From the menu select Format…. Ensure that the option FAT32 (Default) is selected and click Start.

    A few moments later you will see a confirmation that the format has been completed and you SD card is now ready for the next stage.

    Download Image Writer

    In order to flash the SD card in Windows, it is necessary to first download a third-party application named Image Writer. (Here’s the direct SourceForge link for Image Writer)

    Once downloaded, uncompress the .zip file by secondary-clicking it and selecting Extract all from the context-sensitive menu. This will normally place a folder of extracted files into the same folder that contains the downloaded .zip file.

    Flash the SD Card

    Locate and open the folder of uncompressed files that you recently downloaded. Launch the Image Writer application by double-clicking the Win32DiskImager.exe program that you can see in the folder of extracted files.

    If your PC asks you, give permission for the program to make modifications. This will open the Image Writer program.

    In Image Writer click on the blue folder icon, to the right of the white box, and locate your .img file. This will be contained within a folder that you extracted from the downloaded .zip file. Select the .img file and click Open to populate the white box with the path to your image.

    In the Device box, on the right, ensure that the drive letter for your SD card is selected.

    Tip: You can double-check the correct drive letter, for your SD card, by pressing CRTL E to open Windows Explorer and locating the SD card drive.

    When you are satisfied that the path to the .img file and the destination drive letter are both correct, click the Write button to start the process of flashing the SD card. The process will take four or five minutes and you will be shown a green progress bar during the course of the operation.

    Upon completion, a confirmation window will be displayed and you will be free to close Image Writer and eject the SD card before removing it.

    Using the Raspberry Pi

    Now that you've flashed your SD card, go ahead and put it into the slot on your Raspberry Pi. Then you'll want to connect the HDMI cable, Wifi dongle/Ethernet cable, and finally the power cable.

    Booting up

    Once you plug your Pi into a power source, you'll see the boot process on your screen. Once it's done booting up, the pi will ask for your username and password. By default, all Raspbian images have the default username pi and the default password raspberry. Now you can do whatever you want!!! I recommend googling Raspberry Pi Projects and doing something cool. Remember, the more familiar you get with the raspberry pi, the more you will have to use bash (the command line). Don't be afraid to ask others if you have questions on how to do something in bash.
